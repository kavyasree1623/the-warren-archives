# The Warren Archives

A cinematic Conjuring Universe horror experience built with React, TypeScript and Vite.

## Features

* Warren Museum
* Demon Encyclopedia
* Timeline of Evil
* Trailer Theater
* Movie Gallery
* Atmospheric Horror Effects
* Interactive Storytelling

## Tech Stack

* React
* TypeScript
* Vite
* Framer Motion
* Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
