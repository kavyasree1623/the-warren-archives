# The Warren Archives

A cinematic Conjuring Universe horror experience built with React, Vite, Framer Motion and immersive storytelling.

## Features

* Interactive Warren Museum
* Conjuring Universe Timeline
* Demon Encyclopedia
* Artifact Collection
* Trailer Theater
* Movie Gallery
* Flashlight Horror Effects
* VHS Mode
* Atmospheric Audio

## Tech Stack

* React
* TypeScript
* Vite
* Framer Motion
* Tailwind CSS

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
